React Native Interview Activity
===============================

To run the app, make sure you have XCode and/or Android Studio installed and
configured and execute one of the following:

```shell
npm run ios
npm run android
```

To check types, execute:

```shell
npm run ts:check
```

To run the tests, execute:

```shell
npm run test
```

...or with watch:

```shell
npm run test:watch
```
